# Changelog

All notable changes to Tab Guard will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.5] - 2025-12-18

### Fixed
- Updated README, CONTRIBUTING, popup UI, and repository badges to point to [GreatButter888/tabguard](https://github.com/GreatButter888/tabguard)

## [1.0.4] - 2025-12-17

### Added
- GitHub star button in popup UI
- Comprehensive README for public release
- Contributing guidelines (CONTRIBUTING.md)
- Privacy policy document
- MIT License
- Professional repository structure

### Changed
- Enhanced popup UI with better visual design
- Improved accessibility and dark mode support
- Updated Buy Me a Coffee integration

## [1.0.3] - 2025-12-17

### Changed
- Enhanced README with focus and productivity messaging

## [1.0.2] - 2025-12-17

### Added
- Complete redesign of popup UI for Chrome Web Store release
- Stat cards displaying current tab and window counts
- Comprehensive alert system (Info, Warning, Error, Success)
- Loading states with animated spinner
- Real-time form validation with visual feedback
- Buy Me a Coffee button integration

### Improved
- WCAG 2.1 Level AA accessibility compliance
- Enhanced keyboard navigation with visible focus states
- High contrast mode support
- Reduced motion support for users with vestibular disorders
- Screen reader improvements with proper ARIA labels

### Fixed
- Input validation for tab limits (1-9999)
- Error handling with actionable messages

## [1.0.1] - 2025-12-16

### Added
- Buy Me a Coffee image button without remote script
- Content Security Policy for external images

### Changed
- Updated Buy Me a Coffee slug to jacobleecolangelo

## [1.0.0] - 2025-12-16

### Added
- Initial release of Tab Guard
- Tab limiting functionality
- Configurable maximum tab count
- Notification system when limit is reached
- Automatic closing of excess tabs
- Local storage for user preferences
- Extension popup with settings UI

### Features
- Simple tab limiting (1-9999 tabs)
- Smart notifications
- Live statistics (tab and window counts)
- Beautiful UI with dark mode support
- Privacy-first (no data collection)
- Lightweight with minimal performance impact
